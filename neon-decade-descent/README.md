# Neon Decade Descent

A standalone browser roguelike prototype based on the Planning Room test plan.

Open `index.html` in a browser to play. No install step is required.

## Controls

- `WASD` or arrow keys: move
- `.`: wait one turn
- `R`: restart

## Implemented Plan Pieces

- Turn-based roguelike movement on a procedural room-and-corridor map
- Five-floor run structure with win/loss states
- 1980s-themed districts: arcade mall, mixtape subway, cold-war broadcast tower, aerobics studio, video-rental labyrinth
- Era-specific items: mixtape charm, arcade token, floppy disk map, plastic sunglasses, neon keycard, portable cassette
- Era events: market-panic terminal, space-shuttle newsroom, Berlin-wall checkpoint, cable-TV signal storm, mall directory kiosk
- Enemy families, inventory, combat, signal score, fog of war, broadcast log, and responsive UI
